# Introduction

## 开发环境
- os: ubuntu16.04
- 构建工具：cmake
- 编译工具:gnu
- 依赖的库：
    + OpenCV2.0
    + ceres(平差优化)
    + Eigen3(矩阵计算)
    + Gitbook(可选，用于生成api文档)
- 代码管理：Git
